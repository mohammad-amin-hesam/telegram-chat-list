# Finology
Sample landing page for test.

## Installation

Use the package manager [yarn](https://www.npmjs.com/package/yarn) 
or [npm](https://www.npmjs.com/) to install Finology.

### Using npm:
```bash
cd {project_foled}
npm install
npm run build
npm start
```

### Using yarn:
```bash
cd {project_folder}
yarn
yarn build
yarn start
```

#### Open brower in http://localhost:3000
